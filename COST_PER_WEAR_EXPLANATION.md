# ClosetIQ — Cost Per Wear (CPW) Logic

## What is Cost Per Wear?

Cost Per Wear (CPW) = **Purchase Price ÷ Number of Times Worn**

It answers: *"How much did each time I wore this item actually cost me?"*

A ₹45,000 saree worn 3 times = ₹15,000 per wear.  
A ₹1,299 t-shirt worn 50 times = ₹26 per wear.

---

## Rules

### 1. CPW only shows when the item has been worn at least once
- 0 wears → shows **"Not worn yet · target ₹X in Y wears"**
- This prevents misleading low CPW numbers for unworn items

### 2. Target CPW is shown alongside actual CPW
- Target = what CPW would be if worn the expected number of times
- Helps you understand if you're getting good value

### 3. Value Assessment Labels
| Label | Meaning |
|---|---|
| ✅ Great value | Actual CPW ≤ Target CPW |
| ⚡ Wear more | Actual CPW is 1–2× the target |
| 👗 Wear X more times | Actual CPW is >2× the target |

## Examples

### Example 1: Blue Silk Saree (₹45,000)
- Category: Traditional saree > ₹20,000
- Expected wears: **3**
- Target CPW: ₹45,000 ÷ 3 = **₹15,000**
- If worn 1 time: CPW = ₹45,000 → "👗 Wear 2 more times"
- If worn 3 times: CPW = ₹15,000 → "✅ Great value"

### Example 2: Red Cocktail Dress (₹8,999)
- Category: Cocktail dress
- Expected wears: **5**
- Target CPW: ₹8,999 ÷ 5 = **₹1,800**
- If worn 1 time: CPW = ₹8,999 → "👗 Wear 4 more times"
- If worn 5 times: CPW = ₹1,800 → "✅ Great value"

### Example 3: White Cotton T-Shirt (₹1,299)
- Category: T-shirt
- Expected wears: **50**
- Target CPW: ₹1,299 ÷ 50 = **₹26**
- If worn 10 times: CPW = ₹130 → "⚡ Wear more"
- If worn 50 times: CPW = ₹26 → "✅ Great value"

### Example 4: Black Leather Handbag (₹8,999)
- Category: Handbag
- Expected wears: **100** (used almost daily)
- Target CPW: ₹8,999 ÷ 100 = **₹90**
- If used 50 times: CPW = ₹180 → "⚡ Wear more"
- If used 100 times: CPW = ₹90 → "✅ Great value"

## Why This Matters

1. **Expensive ≠ Bad value** — A ₹45,000 saree worn at every family wedding for 10 years is better value than a ₹5,000 dress worn once.

2. **Cheap ≠ Good value** — A ₹500 saree bought for one occasion and never worn again has infinite CPW.

3. **Party/occasion wear has lower expected wears** — It's normal for a cocktail dress to have a higher CPW than a t-shirt. The target accounts for this.

4. **Accessories are high-value** — A ₹12,999 watch used daily for a year has a CPW of ₹36 — excellent value.

## Declutter Suggestions

Items appear in the **Wardrobe Cleanup Suggestions** when:
- Worn 0–2 times AND
- Not worn in the last 30 days

The suggestion helps you decide: wear it more to get value, or declutter it.
